Category A — SaaS/소프트웨어
A-1시장 규모 및 TAM 근거부분 확인▾
[확인] 베트남 11.9M 개인투자자, 80개 증권사, VND 10 quadrillion(GDP 82%) 시장 규모는 IR에서 구체적 수치로 제시됨. [부분 확인] 시장 파편화(fragmented ecosystem) 문제는 명확하나, TAM만 제시되고 SAM/SOM 산정 없어 실제 addressable market 불명확. [미확인] 베트남 증권사들의 자체 앱 고도화, 글로벌 multi-broker 플랫폼(Interactive Brokers 등) 진입 가능성 대비 경쟁 우위 미제시. 시장은 크나 점유 가능성은 검증 필요.
A-2API 통합 기술 장벽 및 차별성미확인▾
[부분 확인] 80개 증권사 API 통합이라는 기술적 과제는 명확히 제시됨. [미확인] 실제 몇 개 증권사와 API 연동을 완료했는지, 파트너십 계약을 체결했는지 데이터 전무. [미확인] 베트남 증권사들이 제3자 플랫폼에 API 제공을 허용하는지, 규제상 제약은 없는지 미검증. [미확인] 기존 증권사 앱 대비 UI/UX, 속도, AI 기능의 실질적 우위를 입증할 사용자 테스트 결과나 retention 데이터 부재. 기술 실행 가능성 높은 리스크.
A-3SaaS 수익모델 및 Unit Economics미확인▾
[미확인] IR 전체에서 수익모델(플랫폼 사용료, 거래 수수료 공유, 증권사 B2B SaaS 라이선스 등) 명시 없음. [미확인] CAC, LTV, churn rate 등 SaaS 핵심 지표 전무. [미확인] '롤아웃' 단계라 하나 실제 active user 수, 거래량, MRR/ARR 데이터 부재. [미확인] 증권사와의 revenue share 협상력, 개인 투자자 대상 유료 전환율 검증 안 됨. 매출 이력 없고 투자 유치 이력 없어 사업 모델 실행 가능성 판단 불가.
A-5확장성(Scalability) 및 지역 확장 전략미확인▾
[미확인] 베트남 외 동남아(태국, 인도네시아 등) 시장 확장 계획 부재. [미확인] 베트남 11.9M 투자자 중 몇 %를 언제까지 확보할 수 있는지 성장 로드맵 없음. [미확인] AI 멀티에이전트, 커스터마이징 기능이 사용자 증가 시 서버·운영 비용 대비 어떻게 스케일하는지 인프라 전략 부재. [미확인] 베트남 단일 시장 의존 리스크(정치·경제 변동성)에 대한 대응 전략 없음. 장기 성장 가능성 불투명.
Category B — 핀테크/금융 서비스
B-1금융 규제 대응 및 라이선스 확보미확인▾
[미확인] 베트남 State Securities Commission(SSC) 규제상 제3자 투자자문·거래 플랫폼 운영에 필요한 라이선스 보유 여부 전혀 언급 없음. [미확인] AI 투자자문(robo-advisor) 제공 시 금융당국 승인 필요 여부, 책임 소재(오류 발생 시 플랫폼 vs 증권사) 미명시. [미확인] 사용자 자금·증권 보관 방식, 보안·개인정보보호 인증(ISO 27001 등) 획득 여부 부재. [미확인] 증권사와의 계약 구조(white-label, API 제공 조건, 수수료 정산) 법적 리스크 검토 안 됨. 금융 플랫폼 핵심 리스크 전면 미확인.
B-3증권사 파트너십 확보 전략미확인▾
[미확인] 80개 증권사 중 몇 개와 실제 파트너십·API 연동 계약 체결했는지 구체적 숫자 없음. [미확인] 증권사 입장에서 자사 고객을 제3자 플랫폼으로 유도할 인센티브(수수료 증대, 신규 고객 유입 등) 제시 부재. [미확인] 대형 증권사(SSI, VNDirect 등)와 중소 증권사 간 협력 우선순위 전략 없음. [미확인] 증권사가 경쟁 플랫폼으로 인식할 가능성, 자체 슈퍼앱 개발로 대응할 리스크 미고려. 핵심 파트너 확보 가능성 심각한 의문.
B-4AI 투자자문 성능 및 신뢰도 검증미확인▾
[부분 확인] AI 멀티에이전트, 5가지 AI Mentor 관점, 리스크·기회 알림 기능은 IR에서 상세 설명됨. [미확인] AI 모델의 투자 성과 백테스트 결과, 예측 정확도, 실제 사용자 수익률 개선 데이터 전무. [미확인] AI 조언의 오류로 인한 손실 발생 시 책임 소재, 사용자 보호 장치(disclaimer, 금융 전문가 검토 등) 부재. [미확인] 베트남 증시 데이터 품질, 실시간성 확보 방안 미명시. AI 기능이 마케팅 레벨인지 실질적 가치 창출하는지 검증 필요.
ANALYSIS SWOT 분석
강점 (Strengths)
  • [확인] 베트남 증시의 파편화 문제는 실재함 — 11.9M 투자자가 80개 증권사에 분산되어 있다는 IR 데이터는 구체적
  • [부분 확인] AI 멀티에이전트 UX는 차별화 요소가 될 수 있으나, 실제 구현 수준·사용자 반응은 미공개
약점 (Weaknesses)
  • [미확인] 증권사 API 통합의 기술적 난이도·라이선스 비용·데이터 보안 규제 대응 방안 부재
  • [미확인] 팀 구성·전문성 정보 전무 — 금융 플랫폼 구축에 필수적인 fintech/보안/규제 전문가 보유 여부 불명
  • [미확인] 수익모델 미명시 — 플랫폼 수수료인지, 증권사 제휴 수익인지, freemium인지 불분명
  • [미확인] 경쟁 우위 근거 부족 — 기존 증권사 자체 앱 대비 락인 요소(network effect, switching cost) 없음
기회 (Opportunities)
  • 베트남 개인투자자 증가 추세 — 디지털 네이티브 세대의 투자 참여 확대 가능성
  • 금융당국의 오픈뱅킹·오픈API 정책 도입 시 시장 진입 장벽 완화 가능
  • AI 투자 조언 수요 증가 — 소액투자자의 정보 격차 해소 니즈
위협 (Threats)
  • 증권사들의 자체 플랫폼 고도화 — 대형 증권사가 직접 AI 기능 탑재 시 경쟁력 약화
  • 금융 규제 리스크 — 투자 조언 라이선스, 데이터 보안, API 접근 제한 등 규제 장벽 높음
  • 글로벌 fintech 플랫폼의 베트남 진출 — Robinhood, eToro 등 자본력 있는 경쟁자 등장 가능성
  • 증권사 파트너십 확보 실패 시 사업 불가 — API 접근권 없이는 플랫폼 구동 불가
