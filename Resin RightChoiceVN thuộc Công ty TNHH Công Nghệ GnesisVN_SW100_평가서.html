A-3경쟁 우위 및 차별화검증 필요▾

[미확인] '6개 제품 라인'이 어떤 응용 분야(dental, jewelry, industrial 등)를 커버하는지 불명확하고, 각 라인의 기술적 차별점(점도, 경화 속도, 강도 등) 명시 없음. 베트남 로컬 브랜드 부재는 기회이나, 기술·품질 경쟁력 입증 없이는 저가 중국 제품과의 차별화 실패 가능성 높음.
A-4비즈니스 모델 확장성중대한 우려▾
[미확인] '레진 판매 + 3D 프린팅 컨설팅 서비스'로 복합 수익 구조 제시하였으나, 컨설팅 매출 비중, 단가, 고객 획득 채널 전혀 불명. [미확인] MakerHub 플랫폼 연계 언급되었으나 MakerHub의 실체(베트남 내 인지도, 사용자 수, 파트너십 조건) 확인 안 됨. [미확인] 2025년 8~12월 목표 매출 1.4억 동이 '레진 판매만' 인지 '컨설팅 포함'인지 불명확. 이익률 50~500% 주장은 비현실적이며, 실제 판매 단가·거래량·마진 구조를 검증할 재무 자료 전무로 모델 신뢰성 매우 낮음.
