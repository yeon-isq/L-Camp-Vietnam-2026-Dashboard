Category A — SaaS/소프트웨어
A-1시장 검증 가능성 (Product-Market Fit)높은 위험 - 검증 불가▾
무료 웹사이트 생성 시장은 Wix, Webflow, WordPress.com 등이 지배하며 freemium 전환율 2-5%가 업계 표준임 [확인]. BAGURU의 차별화 포인트인 '1분 SEO 웹사이트'는 Wix ADI, Webflow AI 등이 이미 구현했으며 [확인], '분산형 고객 교환' 개념은 GDPR/개인정보보호 이슈로 실행 가능성 의문 [미확인]. 베트남 SAM/SOM 데이터 전무로 시장 규모 검증 불가 [미확인]. Telegram 봇 기반 마케팅이 실제 conversion으로 이어지는지 트랙션 데이터 없음 [미확인].
A-2Unit Economics 건전성평가 불가▾
Freemium SaaS의 핵심 지표인 CAC, LTV, payback period, free-to-paid 전환율 데이터 전무 [미확인]. '무료 집 + 유료 가구' 비유는 직관적이나 실제 유료 전환 메커니즘 불명확 [미확인]. 매출 실적 없음이 명시되어 있어 [확인] monetization 가설 검증 전 단계. AI 에이전트 운영 비용(OpenAI API 등 LLM 비용, 인프라)이 freemium 모델에서 감당 가능한지 economics 분석 부재 [미확인].
A-3기술 차별성 및 진입장벽낮음▾
AI 웹사이트 생성은 2024년 이후 commoditized된 기술로 [확인], BAGURU만의 독점적 기술 우위 불명확. '자율 크로스 마케팅 알고리즘'이 핵심 자산으로 주장되나 [부분 확인] 특허 출원, 기술 백서, A/B 테스트 결과 등 검증 자료 전무 [미확인]. Telegram 봇 @TradingAutoVNBot이 lead magnet으로 작동한다는 주장만 있고 [부분 확인] 실제 전환 데이터 미제시 [미확인]. 오픈소스 웹빌더 + 기존 LLM API 조합으로 쉽게 복제 가능한 구조로 보임 [부분 확인].
A-4확장성 (Scalability) 전망구조적 제약 존재▾
Freemium SaaS는 network effect 없이 순수 viral growth에 의존하나, BAGURU의 '분산형 고객 교환'이 실제 네트워크 효과를 창출할지 메커니즘 불명확 [미확인]. 베트남 시장 한정 시 TAM이 제한적이며 글로벌 확장 시 Wix/Webflow와 정면 경쟁 불가피 [확인]. AI 에이전트 품질이 사용자 증가 시 일관성을 유지할지(AI drift, hallucination 관리) 기술적 검증 없음 [미확인]. 현재 'Roll out' 단계로 실제 사용자 수, retention, engagement 지표 전무 [미확인].

D-2팀 구성 및 역량정보 부재 - 평가 불가▾
Self-reported founding team 정보 빈칸 [확인]. CTO만 언급되고 CEO, 제품 리드, 마케팅 담당 등 핵심 포지션 정보 전무 [미확인]. AI/ML 엔지니어링 역량을 증명할 CTO의 이전 경력, 학력, GitHub, 논문 등 확인 불가 [미확인]. SaaS 운영 경험(고객 지원, DevOps, 보안)을 갖춘 팀원 존재 여부 불명 [미확인]. Key-person risk 극대화 - 1인에게 기술/제품/사업 모두 의존하는 구조로 추정 [부분 확인].
D-3실행 트랙 레코드입증 자료 전무▾
'Roll out' 단계라고 명시되나 [확인] 구체적 사용자 수, MAU, retention rate, NPS 등 정량 지표 전무 [미확인]. Telegram 봇 @TradingAutoVNBot이 'proven lead magnet'이라 주장하나 [부분 확인] 획득한 리드 수, 전환율, engagement rate 데이터 없음 [미확인]. 수상 이력, 액셀러레이터 참여, 언론 보도 등 제3자 검증 전무 [확인]. 3년 운영되었다고 하나 이 기간 동안의 피벗 히스토리, 학습 사이클, 제품 반복 과정 설명 없음 [미확인].
